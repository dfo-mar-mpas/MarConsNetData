# Get Geomorphic Unit Data

This function gets the geomorphic unit information of the Scotian Shelf.
This is based on the Greenlaw classification system.

## Usage

``` r
data_geomorphic_units()
```

## Value

sf dataframe object

## Examples

``` r
if (FALSE) { # \dontrun{
geomorph <- data_geomorphic_units()
} # }
```

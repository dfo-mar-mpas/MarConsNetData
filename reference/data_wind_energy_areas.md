# data_wind_energy_areas Download and read the Wind Energy Areas (WEA) dataset from the Canada-Nova Scotia Offshore Energy Regulator [(CNSOER)](https://cnsoer.ca/renewable-energy/lands-management/governments-designated-offshore-wind-energy-areas).

data_wind_energy_areas Download and read the Wind Energy Areas (WEA)
dataset from the Canada-Nova Scotia Offshore Energy Regulator
[(CNSOER)](https://cnsoer.ca/renewable-energy/lands-management/governments-designated-offshore-wind-energy-areas).

## Usage

``` r
data_wind_energy_areas()
```

## Value

An sf object containing the Wind Energy Areas (WEA) dataset.

## Examples

``` r
if (FALSE) { # \dontrun{
wea <- data_wind_energy_areas()
} # }
```

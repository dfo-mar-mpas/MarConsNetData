# Get Biophysical Units Data

This function obtains information (lenght, area, etc.) of smaller
regions in the Scotian shelf that are classified using the "Greenlaw
classification" system. For more information visit Open Data at
https://open.canada.ca/data/en/dataset/2d9cce9a-d634-4b49-879f-87c40c52acf2

## Usage

``` r
data_biophysical_units()
```

## Value

sf dataframe

## Examples

``` r
if (FALSE) { # \dontrun{
biophys <- data_biophysical_units()
} # }
```
